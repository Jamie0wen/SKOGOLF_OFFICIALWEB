# ⛳ Golf Portfolio Website

## Overview
This project is a personal website I am building for my sister, an upcoming elite golfer. The goal of the site is to help grow her online presence, showcase her achievements, and provide a central hub for supporters, sponsors, and followers.

At the same time, this project serves as a way for me to develop and refine my web development skills through a real-world application.

---

## Goals

- Build and grow my sister’s personal brand and following  
- Showcase her golfing achievements, progress, and journey  
- Provide a platform for potential sponsors and supporters  
- Improve my skills in front-end (and potentially back-end) development  

---

## Features

- Responsive design for mobile and desktop  
- Contact form for sponsorship and enquiries  
- Dedicated pages for:
  - About / Profile  
  - Achievements  
  - Sponsorship   
- Clean and modern UI  

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- (Optional/Planned) Form handling via Formspree  

---

## What I Learned So Far

Through this project, I am gaining experience in:

- Structuring and designing a full website  
- Working with forms and handling submissions  
- Debugging real-world issues (e.g., routing, hosting, form responses)  
- Deploying a live site
